## SECTION 4: THREE PRINCIPLES FOR VALUABLE OUTPUT

**Principle 1 — THE SO WHAT RULE:** Every output must tell the reader what to think differently, not just what happened.
- Bad: "MVRV is 1.1"
- Good: "MVRV 1.1 = BTC near realized value, same zone that preceded 3x moves in 2019 and 2023. Constructive but not yet green light."

**Principle 2 — THE CONTEXT LADDER RULE:** Every project-level finding must reference the level above. L6 gem = L5+L4+L3 confirmed. This separates Erasto from Reddit.

**Principle 3 — THE THRESHOLD RULE:** Only alert when a pre-defined quantitative threshold is crossed. Alert fires when MVRV drops below 1.0, not when it drops.
